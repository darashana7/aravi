# Aaravii Construction Website

## Project Overview
Modern, professional website for Aaravii Construction Private Limited, designed for cPanel hosting compatibility.

## Features
- Responsive design (mobile-first)
- Modern, clean aesthetics with construction industry focus
- Interactive contact form with WhatsApp integration
- SEO optimized
- Fast loading with optimized assets
- Accessibility compliant

## File Structure
```
aaravii-construction/
├── index.html          # Main homepage
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
├── images/
│   ├── project1.jpg    # Modern Villa project
│   ├── project2.jpg    # Commercial Complex
│   ├── project3.jpg    # Renovated Office Space
│   ├── project4.jpg    # Luxury Apartment Complex
│   ├── hero-bg.jpg     # Hero background
│   ├── construction-team.jpg
│   └── favicon.ico     # Site favicon
├── robots.txt          # Search engine directives
├── sitemap.xml         # XML sitemap
└── .htaccess          # Apache configuration
```

## Quick Start
1. Upload all files to your cPanel hosting's public_html directory
2. Add images to the images/ folder (see images/index.html for reference)
3. Update contact information if needed
4. Test functionality

## Requirements
- Web server with HTML support
- No server-side processing required (pure HTML/CSS/JS)
- Compatible with standard cPanel hosting
